# 💡 Livnex Consumer Advise App

ระบบคำนวณวงเงินสินเชื่อและเช่าออม สำหรับโครงการบ้านของ SENA Development

## 🔧 Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Data: RefBank + LoanBand (DSR)
## 🖥️ Setup

### 1. ติดตั้ง dependencies
```bash
cd client
npm install
cd ../server
npm install
---
Run System
cd client
npm start
# และ
cd server
npm run dev
